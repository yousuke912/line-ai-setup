STEP 04 — GAS設定

      
GASに貼り付けて設定してください

      
コピーしたコードをGoogleのプログラム実行環境（GAS）に貼り付けます。順番通りに進めれば大丈夫です。

      
    
        
  1. script.google.com を開いて、「新しいプロジェクト」をクリックGoogleにログインしていない場合はログインしてから進んでください
    2. 
        
  2. エディタ内のコードを全選択（Ctrl+A または ⌘+A）→ Delete で削除 → コピーしたコードを貼り付け（Ctrl+V または ⌘+V）
    3. 
        
  3. フロッピーアイコン💾（保存）をクリック
    4. 
        
  4. 上の関数選択欄で「setupLineAI」を選んで ▶ 実行ボタンをクリック → LINEトークンとAPIキーが自動で登録されます「このアプリはGoogleで確認されていません」と出たら → 「詳細」→「（安全ではないページ）に移動」→「許可」で進んでOKです
    5. 
        
  5. 次に関数選択欄で「testAllPermissions」を選んで ▶ 実行 → 権限をすべて許可GmailやGoogleカレンダーなどへのアクセス許可を一括で取得します
    6. 
        
  6. 「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→ アクセス「全員」→「デプロイ」→ 表示されたURLをコピー
    7. 
        
  7. LINE Developers → チャンネル → Messaging API設定 → 「Webhook URL」 にペースト → 保存 → 「検証」ボタンで確認「成功」と出ればOKです
    8. 
        
  8. LINEで「こんにちは」と送信 → AIが返事してきたらセットアップ完了！🎉
    9.
